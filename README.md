========================================
BELEŽKA – NAVODILA ZA UPORABO


1. OPIS
----------------------------------------
Program Beležka je preprost urejevalnik
besedila za delo z .txt datotekami.

Omogoča:
- pisanje besedila
- odpiranje datotek
- shranjevanje datotek
- izbiro kodiranja (UTF-8 / ANSI)
- številke vrstic
- osnovno urejanje besedila

----------------------------------------

2. ZAGON
----------------------------------------
Program zaženeš:
Belezka.exe

Ali z datoteko:
Belezka.exe ime_datoteke.txt

----------------------------------------

3. MENI – DATOTEKA
----------------------------------------
Nova
- ustvari nov dokument

Odpri (Ctrl+O)
- odpre .txt datoteko

Shrani (Ctrl+S)
- shrani dokument

Shrani kot (Ctrl+Shift+S)
- shrani dokument pod novim imenom
- omogoča izbiro kodiranja

Izhod
- zapre program

----------------------------------------

4. MENI – UREDI
----------------------------------------
Razveljavi (Ctrl+Z)
Ponovi (Ctrl+Y)

Izreži (Ctrl+X)
Kopiraj (Ctrl+C)
Prilepi (Ctrl+V)

Izberi vse (Ctrl+A)

----------------------------------------

5. MENI – POGLED
----------------------------------------
Prikaži številke vrstic
- vklop/izklop številčenja vrstic

Širina se prilagaja avtomatsko glede
na število vrstic.

----------------------------------------

6. SHRANJEVANJE IN KODIRANJE
----------------------------------------
Pri "Shrani kot" izbereš:

UTF-8
- univerzalno kodiranje
- podpira vse znake

ANSI (Windows-1250)
- starejši Windows format
- za kompatibilnost s starejšimi programi

Izbrano kodiranje se zapomni.

----------------------------------------

7. STATUSNA VRSTICA
----------------------------------------
Spodaj desno prikazuje:
- trenutno vrstico
- stolpec
- kodiranje

----------------------------------------

8. ODPIRANJE DATOTEK
----------------------------------------
Program najprej poskusi:
1. UTF-8
2. če ne uspe → ANSI

----------------------------------------

9. OPOMBE
----------------------------------------
- Podpira samo tekst (.txt)
- Brez oblikovanja (fonti, slike)
- Brez Word funkcij
